# /slack-update

Turn any document into a concise Slack update for the team channel.

## Instructions

Read the provided document and produce a Slack-formatted update:

1. Start with an emoji and the initiative name + status (e.g., "🚀 Guess the Price — Ready for Review")
2. **TL;DR:** One sentence summary
3. **What we learned:** 3-4 bullet points of key findings or decisions
4. **Recommendation:** What we think we should do
5. **Next step:** What's happening next and by when
6. End with: "Thread: Open for questions/discussion 👇"

## Style

- Max 10 lines total — people skim Slack
- Lead with the conclusion, not the process
- Use bullet points, not paragraphs
- Bold key numbers and decisions
- No hedging — be direct about what you recommend

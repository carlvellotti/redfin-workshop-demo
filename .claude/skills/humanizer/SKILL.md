# /humanizer

Check any document for AI-writing patterns and fix them.

## Instructions

Read the provided document and check for these common AI-writing tells:

**Flag and fix:**
- Opening with "In today's..." or "It's important to note that..."
- Overuse of "leverage," "utilize," "facilitate," "streamline"
- Lists of exactly 3 or 5 items when the content doesn't naturally group that way
- Every paragraph being exactly the same length
- Hedging language: "it's worth noting," "one might consider," "it could be argued"
- Forced transitions: "Furthermore," "Moreover," "Additionally" at the start of every paragraph
- Hollow superlatives: "cutting-edge," "game-changing," "innovative"
- The word "delve"

**Rewrite principles:**
- Vary sentence length. Short sentences hit harder after long ones.
- Use contractions. People write "don't" not "do not."
- Be specific instead of general. Replace "improve engagement" with "get people opening the app 3 more days per week."
- Cut the throat-clearing. If the first paragraph is setup, delete it and start with the second.
- Write like you'd explain it to a smart colleague at lunch.

## Output

For each issue found:
1. Quote the original text
2. Explain why it sounds AI-generated
3. Provide the rewritten version

Then output the full revised document.
